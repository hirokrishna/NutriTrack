# 🥗 NutriTrack

A personal nutrition tracking web app built with pure HTML, CSS, and JavaScript — no frameworks, no backend needed.

## Features

- 🥘 **Tracker** — Tell the AI what you ate, get a full nutrition table (calories, protein, carbs, fat, fibre)
- 📊 **BMI Calculator** — Calculate BMI, BMR, TDEE, ideal weight, and target calories based on your goal
- 📝 **Notes** — Save meal plans, goals, and progress notes
- 🔐 **Login / Register** — User accounts stored locally, data is private per user

## Tech Stack

- HTML + CSS + Vanilla JavaScript
- Google Gemini 2.0 Flash API
- localStorage for user data persistence

## Setup

1. Clone the repo
2. Open `nutritrack.html` in any browser
3. Get a free Gemini API key from [aistudio.google.com](https://aistudio.google.com/app/apikey)
4. Paste your key in code

No installs. No server. Just open and use.
